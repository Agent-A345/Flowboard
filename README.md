## 📊 Flowboard — Intelligent Kanban Task Manager

A modern, fully interactive Kanban Board Web App built using HTML, CSS, and JavaScript, designed for efficient task organization, workflow visualization, and productivity analytics with real-time insights and drag-and-drop interactions.

## 🚀 Features
* **Multi-Board System** – Create, switch, rename, and manage multiple boards for different workflows.
* **Kanban Workflow Management** – Organize tasks into customizable columns like To Do, In Progress, and Done.
* **Drag & Drop Task Movement** – Move tasks seamlessly across columns with smooth drag-and-drop interactions.
* **Task Creation & Editing** – Add, edit, and delete tasks with title, description, priority, due date, and tags.
* **Priority-Based Task System** – Categorize tasks into High, Medium, and Low priority levels.
* **Search & Filtering Engine** – Filter tasks by priority, overdue status, and search queries.
* **WIP (Work-In-Progress) Limits** – Set limits per column to control workflow and avoid overload.
* **Smart Insights Panel** – Displays overdue tasks, stuck tasks, and high-priority pending tasks.
* **Analytics Dashboard** – Shows total tasks, completed tasks, overdue tasks, and average completion time.
* **Overdue & Due Today Indicators** – Highlights tasks with deadlines visually.
* **Tagging System** – Add tags to tasks for better categorization and quick search.
* **Import / Export Functionality** – Import and export board data using JSON files.
